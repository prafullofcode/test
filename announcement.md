# How to Use the GitHub Template

## Step 1: Use Template
1. Go to: https://github.com/aryankaflegithub/protobytes-2.0-template/
2. Click green "Use this template" button
3. Click "Create a new repository"

## Step 2: Name Your Repository
- Name: `protobytes-2.0-team-[TEAMNAME]`
- Make it **Public** (required for judging)
- Click "Create repository"

## Step 3: Add Team Members
1. Go to Settings â†’ Collaborators
2. Click "Add people"
3. Add all team members' GitHub usernames

## Step 4: Clone to Your Computer
```bash
git clone https://github.com/aryankaflegithub/protobytes-2.0-team-[TEAMNAME]
cd protobytes-2.0-team-[TEAMNAME]
```

## Step 5: Start Coding!
- Add your code in `src/` folder
- Commit regularly:
```bash
git add .
git commit -m "Description of changes"
git push
```

## IMPORTANT RULES
- DO NOT modify `.github/workflows/` folder
- DO NOT disable GitHub Actions
- Repository will auto-lock after deadline
- All commits after deadline will be ignored
